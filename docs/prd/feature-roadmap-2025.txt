2. VOLATILITY INDICATORS BY SESSION
   ---------------------------------
   WHY: You show WHEN sessions are active, but not their volatility
        characteristics. Traders need to know BOTH timing AND intensity.
        Research shows average daily range increased 20% in 2025.

   FEATURES:
   - Historical average pip movement per session
   - Real-time volatility meter (current vs. historical average)
   - Color-coded intensity levels:
     * Green: Low volatility (< 50 pips)
     * Yellow: Medium volatility (50-100 pips)
     * Orange: High volatility (100-150 pips)
     * Red: Extreme volatility (> 150 pips)
   - Session comparison: "London Killzone averaging 127 pips today vs. 95 avg"

   DISPLAY OPTIONS:
   - Overlay on existing session timeline chart as intensity bars
   - Dedicated volatility panel below charts
   - Numeric indicators next to session names

   DATA SOURCE:
   - Calculate from historical price data (if available)
   - Integrate ATR (Average True Range) calculations
   - Could use free APIs like Alpha Vantage or Twelve Data

   TECHNICAL IMPLEMENTATION:
   - New data layer in PostgreSQL for volatility metrics
   - Backend API endpoint: /api/volatility/session
   - Enhanced ForexChart.tsx with volatility overlay
   - Update every 15-30 minutes

   USER BENEFIT: Helps traders choose optimal entry times, adjust position
                 sizing based on conditions, avoid overtrading during
                 consolidation

5. CURRENCY CORRELATION MATRIX
   ----------------------------
   WHY: Prevents hidden risk exposure. When EUR/USD and GBP/USD both move
        together (+0.85 correlation), having positions in both means
        concentrated risk. One stop-loss often triggers both losses.

   FEATURES:
   - Real-time correlation heatmap (8x8 grid for major pairs)
   - Correlation scale: -1 (inverse) to +1 (perfect correlation)
   - Color coding:
     * Dark green: Strong positive correlation (+0.7 to +1)
     * Light green: Moderate positive (+0.3 to +0.7)
     * Gray: No correlation (-0.3 to +0.3)
     * Light red: Moderate negative (-0.7 to -0.3)
     * Dark red: Strong negative (-1 to -0.7)
   - Hover tooltips with explanation
   - Historical correlation trends (1 week, 1 month, 3 months)
   - Risk warnings: "WARNING: You have 3 highly correlated positions"

   MAJOR PAIRS TO TRACK:
   - EUR/USD, GBP/USD, USD/JPY, USD/CHF
   - AUD/USD, NZD/USD, USD/CAD
   - EUR/GBP, EUR/JPY, GBP/JPY

   CALCULATION METHOD:
   - Pearson correlation coefficient
   - Rolling 20-day or 50-day period
   - Update every hour or day

   DATA SOURCE:
   - Free forex APIs (Alpha Vantage, OANDA, Twelve Data)
   - Calculate manually from price data

   TECHNICAL IMPLEMENTATION:
   - New component: CorrelationMatrix.tsx
   - Backend calculation service
   - PostgreSQL table: currency_correlations
   - API endpoint: /api/correlation/matrix

   USER BENEFIT: Avoids overexposure to single currency movements,
                 better portfolio diversification, reduces unexpected losses
